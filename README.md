# How to use
1. Install MelonLoader
2. Run Game
3. Drop Mods Folder into Rumble Folder
4. Play Game.
5. Add usernames to UserData/AutomaticInviter/usernames.txt (one per line)
6. Add playfab IDs (found in matchinfo file) to UserData/AutomaticInviter/IDs.txt (one per line)
7. Add usernames to UserData/AutomaticInviter/FriendCodes.txt (one per line)
8. Press the button in the park (located to the right of the friend list) to invite people

- Note: Adds a setting to the park board to automatically invite people when you load into the park.
- - Note note: This can be configured to on by default via ModUI
- Other Note: If you edit the lists while in a park, you must press I on your keyboard to refresh them.
- Another note: inviting by username requires the player to be on your friends list
