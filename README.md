# How to use
1. Install MelonLoader
2. Run Game
3. Drop Mods Folder into Rumble Folder
4. Play Game.
5. Add usernames to UserData/AutomaticInviter/usernames.txt or playfab IDs (found in matchinfo file) to UserData/AutomaticInviter/IDs.txt (one per line)
6. Press the button in the park to invite people

Note: If you update the lists while in a park, you can press I on your keyboard to refresh them.
